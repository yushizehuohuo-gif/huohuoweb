# Styleframe Reset

Date: 2026-07-07
Status: styleframe reset only. Do not implement until explicitly approved.
Scope: homepage visual direction after the failed dashboard / HUD execution.

## 1. Why Current Direction Fails

The current `HuoHuo Field Station` page fails visually.

It is not just unfinished. It is pointed in the wrong visual direction.

Main failure:

- The concept was interpreted too literally. `Field Station` became panels, radar, tokens, scan lines, routes, console blocks, and module labels.
- Those symbols create a "system" look, but not beauty. They feel like a cheap game HUD, internal dashboard, or concept UI mockup.
- The page is visually busy without becoming rich. It has many signs of an operating surface, but no mature composition.
- The hero is heavy. `Nick Huo` is large in a blunt way, not in a designed way.
- `一些低温燃烧的东西` lost its quietness. It became a thick slogan and broke into ugly lines.
- The section rhythm is still mechanical: one required module after another.
- Chinese headings are too big and too repetitive. They make the page feel like PPT section covers.
- The palette is dirty: black, beige, red, cyan, amber, and green compete without a clear visual hierarchy.
- The visual inventory area looks like placeholder blocks. It does not raise the aesthetic level of the page.
- DOTA2, AI tools, player signals, visual experiments, and writing are still labels. They do not become a felt world.

The dashboard version must be treated as a second failed draft.

What to keep:

- Nick Huo / HuoHuoOvO as the identity.
- `一些低温燃烧的东西` as the homepage emotional anchor.
- The deeper idea that Work, Tools, Writing, Visual, and Archive belong to one personal world.
- The name `HuoHuo Field Station` as an internal concept if useful.

What to throw away:

- HUD / dashboard / console visual execution.
- Radar, token walls, dense panels, fake system UI.
- Giant repeated Chinese section headings.
- Heavy dark interface mood.
- Temporary color-block visual studies.
- Requirement-document section sequencing.

## 2. What "Beautiful" Means Here

Beauty here should not be confused with interface density.

For this homepage, beautiful means:

- The first screen has a clear visual center and enough quiet around it.
- Typography feels intentional: scale, weight, line breaks, and bilingual hierarchy are carefully controlled.
- Chinese text feels natural and human, not translated or shouted.
- The palette is restrained and unified. One or two accents are enough.
- The page has breathing room.
- Visual material feels like a real artwork, cover, photograph, poster, scan, or archive object, not a placeholder.
- The visitor can feel a person behind the page before understanding every category.
- Information architecture is present but not loud.
- Work, Tools, Writing, and Visual are arranged like traces of a life and practice, not like SaaS feature modules.
- `一些低温燃烧的东西` appears like an entrance sentence, private title, or signature. It is not a slogan.

The goal:

> First glance: beautiful and memorable.
> Second glance: this is Nick Huo / HuoHuoOvO.
> Third glance: `一些低温燃烧的东西` makes the page feel personal, not templated.

## 3. Three Styleframe Directions

### Direction A: Poster Editorial / 个人海报杂志

#### One-Line Concept

A personal homepage treated as a quiet, high-composition visual poster with a magazine index underneath.

#### First Screen

The first screen is not a dashboard.

It looks like a full-viewport editorial poster:

- large but carefully placed `Nick Huo / HuoHuoOvO`;
- one strong image-like graphic area, not a console;
- `一些低温燃烧的东西` appears as a refined Chinese line, likely below or beside the name;
- a small index of Work / Tools / Writing / Visual appears like magazine contents, not navigation modules;
- the page leaves generous negative space.

Possible composition:

- left side: name and Chinese anchor line;
- right side: one abstract image field, scan, collage, or generated visual;
- bottom: small editorial index with 4-5 entries.

#### Color System

Use a controlled editorial palette:

- warm off-white or smoke paper;
- deep ink or charcoal;
- one restrained red, closer to printed vermilion than UI red;
- optional muted gold/olive as a memory tone;
- no cyan unless it is extremely muted and rare.

The page should not read as dark dashboard. It can use dark ink areas, but paper and air should dominate.

#### Typography and Layout

- `Nick Huo / HuoHuoOvO` can be large, but not brutally heavy.
- Use strong contrast between display English and human Chinese.
- Avoid repeated giant section titles.
- Use editorial captions, indexes, issue numbers, marginal notes, and small headings.
- Chinese line breaks must be manually controlled.
- `一些低温燃烧的东西` should stay on one line on desktop if possible. On mobile, allow two balanced lines only if necessary.

#### How `一些低温燃烧的东西` Appears

It appears as the emotional center, not as a slogan.

Treatment:

- medium-large Chinese text;
- lighter than the current black block;
- enough space before and after;
- no explanation immediately attached;
- helper sentence can appear smaller and lower:
  `玩家信号、AI 工具、视觉实验、个人笔记，和一些还没找到容器的东西。`

#### Work / Tools / Writing / Visual Organization

Below the poster:

- Work becomes a short editorial index, maybe 3 entries.
- Tools become a marginal tool list, not cards.
- Writing becomes short fragments in a column.
- Visual becomes one or two high-quality image blocks, not a grid of placeholders.
- Archive is a quiet link, not a CTA module.

#### Why It Is More Beautiful Than Dashboard

It removes fake interface decoration and replaces it with composition, type, spacing, and one strong visual object.

It lets the page feel designed before it feels explained.

#### Risks

- If the visual image is weak, the page may become too plain.
- If typography is not tuned carefully, it may look like a generic editorial portfolio.
- It requires stronger restraint than the current implementation.

### Direction B: Soft Archive / 余温档案

#### One-Line Concept

A warm, private archive of public-safe traces: work, tools, notes, visual fragments, and things that still have residue.

#### First Screen

The first screen feels like an archive table or paper folder, not a control room.

It may include:

- soft paper background;
- one image or scan-like object;
- `Nick Huo / HuoHuoOvO` as a quiet identity mark;
- `一些低温燃烧的东西` as the main entrance sentence;
- small dated labels or archive marks;
- 2-3 fragments arranged with asymmetry.

The page should feel touched, saved, and organized, but not nostalgic in a cheap way.

#### Color System

- warm paper;
- graphite gray;
- faded black;
- one low red accent;
- muted amber for memory;
- almost no bright colors.

The overall image should feel warm and quiet, not beige-template. Texture must be subtle.

#### Typography and Layout

- Chinese text is central and carefully spaced.
- English can become annotation, not headline.
- Use smaller, quieter headings.
- Use archive slips, captions, labels, and short fragments.
- Avoid big blocky titles.
- Avoid visual hierarchy that feels like a dashboard.

#### How `一些低温燃烧的东西` Appears

It becomes the first sentence the visitor wants to read.

Treatment:

- elegant Chinese line;
- placed near Nick's name or on a paper/image surface;
- not too bold;
- not surrounded by literal "burn" graphics;
- no explanatory paragraph directly after it.

#### Work / Tools / Writing / Visual Organization

- Work appears as archive entries: Boommunity, DOTA2 community work, signal workbench.
- Tools appear as instruments in a small ledger: Obsidian, Codex, Hermes, ComfyUI, Figma.
- Writing appears as note slips.
- Visual appears as image fragments, scans, generated studies, or contact-sheet pieces.
- Archive is the main organizing metaphor.

#### Why It Is More Beautiful Than Dashboard

It changes the emotional temperature.

Instead of performing "system intelligence", it lets Nick's contradictions sit together: official work and private writing, AI tools and human residue, DOTA2 memory and future workflows.

It is more personal, softer, and more capable of holding `一些低温燃烧的东西`.

#### Risks

- It can become too quiet if no strong image exists.
- It can drift into generic "nice paper archive" if DOTA2 / AI / tool signals are too hidden.
- It requires careful selection of visual fragments so the page does not become a scrapbook.

### Direction C: Personal Index with One Strong Image / 单一主视觉个人索引

#### One-Line Concept

One powerful personal visual carries the first impression; everything else becomes a light index.

#### First Screen

The first screen is built around a single strong image or image-like composition.

Possible visual object:

- a generated abstract image with game-map memory and paper texture;
- a collage of visual fragments;
- a scanned poster-like field;
- a custom symbol for `HuoHuoOvO`;
- a cover image that feels like a private magazine issue.

Text is minimal:

- `Nick Huo / HuoHuoOvO`;
- `一些低温燃烧的东西`;
- one small helper line;
- a quiet index.

#### Color System

The color system is extracted from the strong image.

Rule:

- image defines the palette;
- page uses mostly neutrals;
- accent is limited to one red or amber;
- no separate dashboard accent system.

#### Typography and Layout

- Typography should support the image, not compete with it.
- Fewer headings.
- More caption-like writing.
- Use a cover / index / footnote logic.
- Let the image occupy 45-65% of the first screen on desktop.

#### How `一些低温燃烧的东西` Appears

It appears as the title of the visual object or the subtitle of the cover.

It should feel like the name of an issue, exhibition, or private collection.

#### Work / Tools / Writing / Visual Organization

- Work is a compact index.
- Tools are footnotes or a tool shelf.
- Writing is 2-3 fragments.
- Visual has one large image and a small set of supporting studies.
- Archive remains a link, not a module.

#### Why It Is More Beautiful Than Dashboard

It restores focus.

The page stops trying to visualize every concept at once. One strong image creates memory; the rest of the page can be light.

#### Risks

- The whole direction depends on the quality of the main image.
- If the image is too generic AI art, the site becomes less personal.
- If the image is too abstract, visitors may not understand Nick's game/community/tool background.

### Direction D: Quiet Game Memory / 安静的游戏记忆

A personal homepage that borrows the emotional memory of long-running games without using game UI.

#### First Screen

The first screen feels like a quiet poster for an old game community:

- old paper;

- no bright cyan;

#### Typography and Layout

#### How `一些低温燃烧的东西` Appears

## 5. Why This Direction Fits Nick

Nick is not best represented by a fake operating system or a literal dashboard.

The strongest identity is not "he has many tools"; it is:

- he reads player emotion;
- he works around long game-community memory;
- he uses AI tools to move judgment earlier;
- he keeps visual experiments and unfinished writing in the same life;
- he does not want a single job-label homepage.

Direction C fits because:

- It gives the site a first-glance visual memory.
- It avoids explaining every part at once.
- It makes `一些低温燃烧的东西` feel private and intentional.
- It can hold DOTA2, AI tools, writing, and visual experiments without turning them into dashboard modules.
- It can still grow into an archive later.

Why not Direction A alone:

- Poster Editorial is beautiful, but it may become too design-portfolio-like if the main visual is not personal enough.

Why not Direction D alone:

- Quiet Game Memory is rich, but it risks over-centering the game layer and reducing the AI/tool/workflow side.

The best path is:

> One strong visual cover from Direction C, softened and humanized by the archive logic of Direction B.

## 7. Visual Rules

### Color

Use a smaller palette:

- paper: warm off-white, not beige-heavy;
- ink: deep charcoal, not pure black;
- red: one muted burn red;
- amber/olive: optional memory accent;
- gray: only for captions and rules.

Remove:

- bright cyan;
- multiple UI accent colors;
- dirty black/green dashboard surface;
- red/cyan/gold competition.

### Typography

Rules:

- `Nick Huo / HuoHuoOvO` can be large, but must not be brutal.
- Chinese anchor line must be set with care and air.
- Avoid giant repeated section titles.
- Section labels should be small, editorial, and sparse.
- Use weight contrast, not only size.
- English should often behave as annotation.

`一些低温燃烧的东西`:

- not ultra-bold;
- not all squeezed;
- not broken into ugly lines;
- ideally one line on desktop;
- on mobile, break as:
  `一些低温燃烧的`
  `东西`
  only if the spacing is beautiful.

### Space

- More negative space.
- Fewer boxes.
- No dense panel walls.
- Let the main visual breathe.
- Let the content sit like an index.

### Graphics

Use:

- one strong image or graphic;
- subtle paper or scan texture;
- small editorial rules;
- captions;
- numbering if useful.

Do not use:

- radar;
- HUD;
- terminal;
- token pills;
- fake operating system;
- dashboard panels;
- generic bento grid.

### Images

Images are mandatory for the next visual reset.

Acceptable image directions:

- generated abstract cover image;
- DOTA memory color field;
- visual collage;
- scan-like paper/image study;
- poster texture;
- personal symbol.

Unacceptable:

- empty CSS blocks;
- generic AI gloss;
- dark blurry stock images;
- placeholder gradients pretending to be art.

### Chinese Typesetting

- Manually control line breaks for the core Chinese sentence.
- Avoid machine-translated bilingual duplication.
- Keep Chinese sentences short and natural.
- Do not make every Chinese heading huge.
- Use Chinese as emotional structure, not only labels.

### Motion

Motion should be almost invisible:

- slow image fade;
- gentle text reveal;
- small hover on index rows;
- no scan lines;
- no pulsing active tokens;
- no dashboard motion.

Reduced motion must preserve the same composition.
